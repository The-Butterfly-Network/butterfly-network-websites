Server Rules

Guidelines for our Minecraft community

Welcome to our Minecraft server! Please read and follow these rules to ensure everyone has a great time building and exploring together.
0

Discord Server Rules Apply

All Discord server rules still apply here! This server is for paid SVG members only.

  • Discord Server Rules still apply!
  • Paid SVG Members ONLY!
1

Be Respectful

Treat all players with kindness and respect, regardless of their skill level or background.

  • No harassment, bullying, or discrimination, in any form, including, but not limited to racism, xenophobia, transphobia, homophobia, ableism, pluralphobia, sexism, antisemitism, islamophobia, fatphobia, or classism
  • Keep chat language appropriate for all ages
  • Respect others' builds and personal space
  • Help new players learn the ropes
2

No Griefing or Stealing

Respect other players' hard work and creativity. Build your own amazing creations!

  • Don't destroy or modify others' builds without permission
  • No stealing from chests, farms, or storage areas
  • Ask before taking resources from community areas
  • Report any griefing you witness to MC server team
3

Fair Play Only

Keep the game fun and fair for everyone by playing without unfair advantages.

  • No hacking, cheating, or exploiting bugs
  • No X-ray mods or unfair resource packs
  • Approved client mods only (ask MC server team if unsure)
  • No duplication glitches or item exploits
4

Build Responsibly

Create amazing builds while being mindful of the server and other players.

  • No inappropriate builds or symbols
  • Keep large builds away from spawn area
  • Don't build too close to others without permission
  • Clean up test builds and temporary structures
5

Follow MC Server Team Instructions

Our moderators and admins are here to help make the server enjoyable for everyone.

  • Listen to and follow MC server team member instructions
  • Don't argue with MC server team decisions in public chat
  • Contact MC server team privately if you have concerns
  • Report rule violations to MC server team members
⚠️

Consequences

Breaking these rules may result in warnings, temporary bans, or permanent removal from the server.

  • First offense: Warning and explanation
  • Minor violations: Temporary jail
  • Serious violations: Temporary ban (1-7 days)
  • Severe/repeated violations: Permanent ban

Questions about the rules? Need help with something?

Contact our MC server team in-game or on Discord!
